## 关键技术事实

- pixi-live2d-display 必须用 **lipsyncpatch fork**（0.5.0-ls-8）才兼容 PixiJS v7，且自带 `model.speak()` 口型同步。
- Tauri capabilities 的 `windows` 必须含窗口标签 `companion`，否则 event.listen 被拒。
- ureq 3.x：超时走 Agent config；`send_json` 需 `json` feature；rustls 需手动 `install_default()`（msedge-tts 共存时）。
- keyring 用 3.x（4.x API 大改）。
- cargo 双 bin 需 `default-run = "app"`，否则 tauri dev 失败。
- whisper-cli（brew whisper-cpp）+ `~/.desktop-assistant/models/ggml-base-q5_1.bin`；只吃 16kHz wav（cpal 录音后线性重采样）。
- 声线绑定落点：`character.manifest.yaml` 无 voice 时按 gender 回落（女=XiaoyiNeural/三月七，男=YunxiNeural/丹恒）；`store.onChanged → tts.setVoice`。
