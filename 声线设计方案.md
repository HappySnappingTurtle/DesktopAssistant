## 设计原则

1. **主力：GPT-SoVITS**（AISHELL-3 开源数据集训练 + 社区优质参考音频）
2. **声线必须匹配角色性格**——不是随便分配，而是有依据的选择
3. **Edge TTS 作为在线备用**（GPT-SoVITS 服务未启动时回落）
4. **女性角色优先从社区获取高质量参考音频**（不仅依赖 Edge TTS 合成）

## GPT-SoVITS 训练方案

### 数据来源

**训练数据：AISHELL-3**
- 218 位说话人（176 女 + 42 男），每人有足够的训练数据
- 16kHz 高保真录音，标注准确率 >98%
- 学术免费下载（openslr.org/93）
- GPT-SoVITS 只需 1 分钟音频即可训练

**参考音频：多源获取**
- AISHELL-3 说话人片段（训练同源，效果最佳）
- GPT-SoVITS 社区分享的参考音频包
- 开源语音数据集（Common Voice、DiDiSpeech 等）
- 女性角色优先从 B 站/GitHub 社区寻找符合性格的音色样本

### 训练流程（每角色 ~2 分钟 GPU 时间）

```
1. 从 AISHELL-3 选取匹配说话人（按性别+年龄+音色筛选）
2. 提取该说话人 1-5 分钟音频片段
3. 准备参考音频（3-10 秒清晰语音 + 对应文本）
4. GPT-SoVITS 训练（SoVITS 8 epochs + GPT 15 epochs）
5. 导出模型文件（GPT ~150MB + SoVITS ~80MB）
6. 放入角色包 voice-models/<char-id>/
7. 更新 character.manifest.yaml 的 gpt_sovits 字段
```

### 说话人选择标准

| 角色性格 | 选择标准 | 候选 AISHELL-3 说话人特征 |
|---|---|---|
| 活泼猫系（F1 小猫） | 年轻、语速偏快、音调高 | 20-25 岁女性 |
| 温柔安静（F2 Hiyori） | 轻柔、语速慢、高音细腻 | 20-28 岁女性 |
| 傲娇猫耳（F3 White） | 成熟、语速稳定、中低音 | 25-35 岁女性 |
| 沉稳专业（M1 绅士） | 低沉、节奏感强、磁性 | 30-40 岁男性 |
| 随和温暖（M2 邻家少年） | 清朗、平和、不高不低 | 25-30 岁男性 |
| 安静内敛（M3 清风少年） | 平和、语速慢、沉稳 | 20-28 岁男性 |

### 参考音频要求

- 格式：WAV 16kHz 16bit 单声道（或高质量 MP3）
- 时长：3-10 秒
- 内容：清晰中文语句，无背景噪音
- 配套：必须提供对应文本（ref_text）
- 情绪：匹配角色性格基调
